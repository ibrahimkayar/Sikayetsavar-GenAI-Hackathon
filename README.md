# 🚀 ŞİKAYETSAVAR – GenAI Marketing Hackathon Project

A Generative AI-powered marketing assistant developed for the **Samsung Innovation Campus Generative AI Hackathon**.

ŞİKAYETSAVAR analyzes customer complaints, identifies the underlying issue, converts negative feedback into marketing opportunities, generates campaign ideas, and creates AI-powered advertisement poster concepts.

---

# 🌟 Project Overview

Instead of treating customer complaints only as problems, ŞİKAYETSAVAR helps brands transform negative customer experiences into meaningful marketing strategies.

The application performs the following workflow:

```
Customer Complaint
        │
        ▼
Complaint Analysis
        │
        ▼
Marketing Opportunity
        │
        ▼
Campaign Generation
        │
        ▼
Poster Concept
        │
        ▼
AI Advertisement Background
```

---

# ✨ Features

- 🔍 Customer complaint analysis
- 😊 Sentiment analysis
- 📂 Complaint categorization
- ⚠️ Severity detection
- 💡 Marketing opportunity generation
- 📝 AI-generated campaign texts
- 🎨 Advertisement poster concept generation
- 🛡️ Ethical AI safeguards for sensitive complaints

# ▶️ Run Locally

### Prerequisites

- Node.js

### 1. Install dependencies

```bash
npm install
```

### 2. Configure API Key

Create a `.env.local` file.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### 3. Run the application

```bash
npm run dev
```
